# Carpeta de Imágenes - NIKA

Aquí es donde debes guardar todas las imágenes del proyecto. Los archivos que necesitas subir son:

## Imágenes Requeridas

| Archivo | Descripción | Dimensión recomendada |
|---------|-------------|----------------------|
| `hero.jpg` | Imagen principal (hero) - niña sonriendo aprendiendo | 700x800px |
| `classroom.jpg` | Imagen de clase / niños en el aula | 600x400px |
| `reading.jpg` | Imagen de niños leyendo / actividad educativa | 300x300px |
| `learning.jpg` | Imagen de aprendizaje lúdico / juegos | 300x300px |
| `enrollment.jpg` | Imagen de inscripción / niño feliz en clase | 500x400px |
| `portrait.jpg` | Retrato / fotografía de perfil | 100x100px |

## Cómo Usar

1. Guarda cada imagen en esta carpeta con el nombre exacto indicado
2. Los formatos aceptados son: `.jpg`, `.png`, `.gif`, `.webp`
3. El proyecto automáticamente usará estas imágenes en lugar de las imágenes de demostración

## Nota

- Las imágenes se referencian como `/images/nombre-archivo.extension`
- Si cambias los nombres, también deberás actualizar los componentes
- Asegúrate que las dimensiones sean similares a las recomendadas para una mejor visualización
